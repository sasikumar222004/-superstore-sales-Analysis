# 🛒 Superstore Sales Performance Dashboard

## 📌 Project Overview
Analyzed 9,994 Superstore sales records using 
MySQL, Excel & Power BI to build an interactive 
Sales Performance Dashboard.

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| MySQL | Data extraction & SQL queries |
| Excel | Pivot Tables & Data cleaning |
| Power BI | Interactive Dashboard & DAX |

## 📊 Dashboard KPIs
- 💰 Total Sales: **$2.27M**
- 📈 Total Profit: **$282.86K**
- 🎯 Profit Margin: **12.47%**
- 📦 Total Orders: **5K**
- 🛍️ Avg Order Value: **$229.87**

## 🔍 Key Insights
- ✅ **West region** — Highest sales ($725K)
- ✅ **Technology** — Top category (36.78%)
- ✅ **Sales grew 52%** from 2014 to 2017
- ✅ **Furniture/Tables** — Negative profit (loss!)
- ✅ **Consumer segment** — Highest orders

## 📁 Files
